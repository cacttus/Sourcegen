# Sourcegen
WPF source generating application for projects.
Generates:
* Java classes
* C++ classes
* C++ H/CPP files
* Qt classes

Configurable with:
* name,
* copyright, 
* license, 
* cryptic ifguard

Automatic BSD license included.
